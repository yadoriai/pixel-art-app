
# pixel art app

32 × 32 マスのキャンバスにドット絵を描ける、シンプルなピクセルエディタです。  
ペン・消しゴム・範囲塗りつぶし、色相環からの色選択、Undo/Redo、PNG 保存に対応しています。  
PC とスマホの両方から使えることを意識して設計しています。

## 主な機能

- 32 × 32 ドットキャンバス
  - 常時表示のグリッド
  - 保存時はグリッド無しの PNG を出力
- 描画ツール
  - ペンツール
  - 消しゴムツール
  - ドラッグで矩形を指定して塗りつぶす範囲ツール
  - 範囲ツール使用中は点線枠で選択範囲をプレビュー
- カラー機能
  - iro.js による色相環＋明度スライダー
  - 選択した色を保存できるスロット（8 個）
  - スロットをクリックして色の再選択が可能
- Undo / Redo
  - 描画操作・範囲塗りつぶし・リセットを 1 アクションとして履歴管理
  - 履歴に基づく Undo / Redo ボタン
- ファイル保存
  - 現在のキャンバスを PNG 形式でダウンロード
- レイアウト・UI
  - PC ではキャンバス＋サイドバーの 2 カラム
  - スマホではキャンバス下にツールをまとめたレイアウト
  - ヘッダーに Undo / Redo / 保存 / リセットボタン
  - ドット調フォントによるタイトル表示
  - カラーテーマ
    - 全体背景: `#7ebeab`
    - ヘッダー背景: `#a8ffd3`
    - タイトル文字: `#798a84`

## 使用技術

- Backend
  - Python 3
  - Flask
- Frontend
  - HTML5 / CSS3
  - JavaScript (素の JS)
  - Canvas API
  - iro.js（カラー ピッカー）
  - Lucide Icons（ボタン用アイコン）
- フォント
  - x12y16pxMaruMonica（環境に応じて変更）
  - フォールバックとして DotGothic16

フォントのライセンスは配布元の規約に従ってください。

## セットアップ

前提: Python 3 系と `pip` がインストールされていること。

```bash
git clone <このリポジトリのURL>
cd <クローンしたディレクトリ>

# 仮想環境の作成と有効化 (任意だが推奨)
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate

# 依存ライブラリのインストール
pip install -r requirements.txt
# requirements.txt が無い場合は少なくとも
# pip install Flask
```

`requirements.txt` の例:

```text
Flask>=3.0.0
```

## 起動方法

```bash
# 仮想環境を有効化した状態で
flask --app app run
# もしくは
python app.py
```

ブラウザで下記にアクセスします。

```text
http://127.0.0.1:5000/
```

## 使い方

1. キャンバスのグリッド上をドラッグして、ペンツールで描画します。
2. 消しゴムツールを選択すると、背景色で上書きして消去します。
3. 範囲ツールを選択し、ドラッグで矩形を選ぶと、その範囲を現在の色で塗りつぶします。
4. 色相環で色を選び、気に入った色をスロットに登録します。

   * 登録済みスロットをクリックすると、その色を再利用できます。
5. Undo / Redo ボタンで直前の操作を戻す／やり直します。
6. 保存ボタンを押すと、グリッド無しの PNG 画像をダウンロードします。
7. リセットボタンでキャンバス全体を白に戻します。
